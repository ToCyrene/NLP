# NLP
《自然语言处理》结课作业

# 代码说明
tune.py             Qwen3 微调代码（我的工作）
model_sever.py      模型主体推理代码（我的工作）
rules_generated.py  数据挖掘得到的规则，用于静态测试
rules.py            数据挖掘得到的规则
analyzer.py         静态测试的主体代码
app.py              前端代码 （部分是我的工作）
run.sh              在服务器上运行工具的脚本（我的工作）
kill.sh             在服务器上终止工具的脚本（我的工作）


# 运行要求
该工具要求gpu显存至少为28G，代码目前的逻辑是要求有三张gpu，可以通过简单的修改适配其它硬件环境。
该工具要求获取llama3 7b和Qwen3 7b的模型权重，也可以使用deepseek r1 7b来代替其中之一。
该工具为C/S架构，主体部署在服务器上，并可以在同一局域网下的任何其它设备进行访问，并配有成熟的前端。


# 运行过程
首先取得至少两个模型的的原始权重，并进行训练，取得lora层。
然后修改model_sever.py里的路径，使其指向上述所有权重。
然后在服务器上运行run.sh，此时工具在服务器上运行。

然后便可以在服务器上运行工具。
先访问SSH。以下是客户端的SSH运行命令：
ssh -p 17022 -l xxxx -L 7899:127.0.0.1:7893 172.28.9.53

保留上述SSH终端，在浏览器中访问 http://localhost:7899/ 即可使用工具。
